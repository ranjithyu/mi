Welcome to my sample HTML page

This is just a simple example to demonstrate basic HTML structure and styling.

You can modify and expand it as you like to create your own web pages.

Here are some key HTML elements:

  • <html>: The root element of an HTML page
  • <head>: Contains meta-information about the document
  • <title>: Sets the title of the document
  • <body>: Contains the visible content of the document
  • <h1>, <h2>, <h3>...: Headings of different levels
  • <p>: Paragraphs of text
  • <ul>, <ol>: Unordered and ordered lists
  • <li>: List items
